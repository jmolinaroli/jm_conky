Installation
------------

Download the configurations files from GitHub repository.

Copy in /usr/bin/:
* spotify-nowplaying
* spotify-nowplaying-length
* spotify-nowplaying-work

Copy in /home/USER/:
* .conky
* .conkyrc0
* .conkyrc1
* conkystart.sh

Otherwise you can use the configuration file:

```sh
$ ./install.sh
```

To start the scripts use conkystart.sh

Enjoy!

Features
--------

* clock
* CPU graph
* memory graph
* hdd graph
* network info
* battery graph
* date/spotify
